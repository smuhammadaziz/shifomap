# Console Web — Developer Dashboard

Admin/developer dashboard for managing users, clinics, and platform analytics.

## Run the app

From this folder (`apps/console-web`):

```bash
pnpm dev
```

The app runs at **http://localhost:4000**.

- **Login:** `admin` / `admin1234`

## Other commands

- `pnpm build` — production build
- `pnpm start` — run production server (port 4000)
- `pnpm lint` — run ESLint
